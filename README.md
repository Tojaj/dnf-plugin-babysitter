dnf-plugin-babysitter
=====================

Do not let the dnf to remove the most precious thing you have!

Installation
------------

Copy `babysitter.py` and `babysitter.conf` to the `/usr/share/dnf/plugins` directory.

Requirements
------------

dnf >= 0.4.10 (I guess)
